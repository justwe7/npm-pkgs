# @justwe7/stylelint-order-standard

## Installation
```bash
npm i @justwe7/stylelint-order-standard -D
```

## Usage
Set your stylelint config to:
```json
{
  extends: [
    '@justwe7/stylelint-order-standard'
  ]
}
```
